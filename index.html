resourced (express.js)

A REST/resource-oriented DSL for configuring koa which makes it easy to defined HTTP resources.

Usage

The following a simple resource which responds to GET requests:

module.exports = new Resource({
    url: "/person/:id",

    cache: cache.minutes(5).publically(),

    respondsTo: [
        http.get(function * (id) {
            ensure(id).populated().numeric({ min : 0 });

            var person = // potentially yield whilst loading from data store or external service

            person.address = this.urlFor(addressResource, { id: person.addressId });

            return person;
        })
    ]
});
